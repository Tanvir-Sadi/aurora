# Aurora Video Asset

Place your realistic aurora loop here for video-first rendering:

- `assets/aurora-loop.webm` (preferred for size)
- `assets/aurora-loop.mp4` (fallback)

Recommended media settings:
- Duration: 12-30 seconds seamless loop
- Resolution: 1280x720 or 1920x1080
- Frame rate: 24-30 fps
- Bitrate target: 1.5-4 Mbps
- Visual style: dark sky, wispy diagonal aurora curtains

If no video file is available, the app automatically falls back to `aurora.js` canvas rendering.
