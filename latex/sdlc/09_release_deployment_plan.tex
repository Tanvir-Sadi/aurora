\section{Deployment Strategy}
\begin{itemize}[leftmargin=*]
  \item Deployment model: static hosting.
  \item Build output: static HTML/CSS/JS assets.
  \item Candidate targets: GitHub Pages or Netlify.
\end{itemize}

\section{Pre-Release Checklist}
\begin{itemize}[leftmargin=*]
  \item Test cases TC-01 to TC-10 executed and recorded.
  \item No open critical or high defects.
  \item Lighthouse performance and accessibility checks captured.
  \item Reduced-motion behavior verified.
  \item SDLC docs synchronized with final implementation.
\end{itemize}

\section{Rollback Plan}
\begin{itemize}[leftmargin=*]
  \item Keep previous deployment artifact available.
  \item Revert DNS/hosting alias to prior stable version if needed.
  \item Publish rollback note with issue summary.
\end{itemize}

\section{Go-Live Validation}
\begin{itemize}[leftmargin=*]
  \item Open production URL on Chrome, Edge, Firefox.
  \item Validate activation flow and animation visibility.
  \item Validate keyboard and reduced-motion paths.
\end{itemize}
